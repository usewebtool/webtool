# webtool

A CLI for your browser.

## Requirements

- Chrome 144 or newer is required.
- In Chrome, open `chrome://inspect/#remote-debugging` and enable remote debugging for your current browser session before using `webtool`.

## Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `WEBTOOL_CHROME_DATA_DIR` | OS default | Chrome user data directory for DevToolsActivePort discovery |
